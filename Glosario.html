Glosario

Palabras a definir

Internet

El termino "internet" se refiere a un grupo grande de computadoras conectadas unas de las otras a través de cables, lineas telefónicas y otros tipos de cableado tecnológico

La web

Es el universo de la información accesible a través de la red o internet.

Tim Barners -Lee

Fue uno de los inventores del World Wide Web junto a Robert Cailiau y es conocido como el padre de la World Wide Web. El redacto el HTML estableciendo los enlaces con otros documentos en una computadora y elabora un esquema que da a cada página de la red un URL

Servidor

Es una computadora que se especializa en almacenar datos e información.Es típicamente una computadora poderosa con una capacidad de almacenamiento superior a las computadoras personales.

URL

El Uniform Resource Locator,es la direction web, el texto que especifica donde se puede encontrar un recurso en internet y su mecanismo para hacedero.

Protocolo

le indica al browser que protocolo debe usar.Un protocolo es un conjunto de métodos o reglas para intercambiar o transferir datos a través de una red de computadoras.

Dominio

indentifica el nombre de la maquina servidor de destino desde el cual se puede descargar el archivo.

File Path

es la ruta virtual al archivo o recurso en el web server o dominio nombrado, incluidos los nombres de directorios principales aplicables.

Link

un link es una cadena de texto vinculada a un URL que utilizamos para poder saltar con facilidad de un documento a otro de forma instantánea.

HTML

Hyper Text Markup Language,fue inventado por Tim Barners-Lee;es un lenguaje de computadora que define la estructura y la presentación
